# Final_Assignment3_Game
Assignment #3 Instructions (3 parts)
Building a 2D Game Vertical Slice (worth 45% total)
The final assignment is to design and build a 2D Game Vertical Slice that will serve to demonstrate your team's understanding of the topics explored throughout the course.

A vertical slice is a more polished and complete version of a game than a prototype and is used to showcase what the final game will look and play like. It often includes all the final game mechanics, systems, and features.

This assignment consists of three components, each with their own delivery date.

Milestone One of Assignment #3 : (worth 10%) 
In this deliverable, the vertical slice should showcase the core mechanics of the game and give a general sense of what the final product will look and feel like. Include a proposal document that details the game’s mechanics. Students should include at least one level/arena/map in their vertical slice.

The Proposal Document Must Include:
Game Concept: Describe the game concept and its objective. What is the player’s goal? What is the theme of the game? [3 marks]
Game Mechanics: List the game mechanics that the player will use to achieve the game objective. For each game mechanics, explain how it works and how it might be implemented in Unity. [5 marks]
Technical Feasibility: Explain the technical requirements of your proposal. How will you implement the listed mechanics? Refer to techniques explored throughout the course. [5 marks]
Conclusion: Write a summary and explain why it is an engaging and enjoyable game. [2 marks]
Milestone Two of Assignment #3: (worth 15%)
In this deliverable, the vertical slice should include all the core mechanics, levels, and characters that you plan to include in the final game along with a functioning UI. Include a revised proposal document that highlights the changes that have been made to the vertical slice because of your playtesting at the conclusion of the previous milestone. Include a playtesting summary report that compiles the results of that playtesting. This submission should also have basic sound effects and music.

Milestone Three of Assignment #3: (worth 20%)
In the deliverable, the vertical slice should be complete and polished and include all the features and mechanics outlined in the final version of the proposal, which should highlight the changes that have been made because of your playtesting at the conclusion of the previous milestone. Include a playtesting summary report that compiles the results of that playtesting. Additionally conduct a post-mortem report that analyzes the game’s development process, identifies any challenges you faced, and discusses the strengths and weaknesses of the final product.
